Overview

The Network Multi-Tool is a Python-based project providing essential network utilities for scanning, testing, and troubleshooting network connections. It includes tools such as a network scanner, packet sniffer, WHOIS lookup, traceroute, and ping utility.

Installation

  1. Clone the repository: 
    git clone https://github.com/yourusername/network-multi-tool.git
  2. Navigate to the project directory: 
    cd network-multi-tool
  3. Install the required dependencies: 
    pip install -r requirements.txt

Usage

Run the main script to start the tool:

python main.py

Choose a feature from the interactive menu:

  • Network Scanner
  • Packet Sniffer
  • WHOIS Lookup
  • Traceroute
  • Ping

Modules

The project is organized into the following modules:

  • core/: Contains the main functionality, including network scanning, traceroute, and more.
  • utils/: Helper scripts for logging, configuration, and common tasks.
  • ui/: Handles user interfaces, including CLI and future GUI enhancements.

Tests

To run the tests, use:

python -m unittest discover tests

The test suite includes:

  • Tests for core functionality (e.g., scanner, traceroute).
  • Validation of utility functions (e.g., hostname resolution, IP validation).
