Introduction

Le Network Multi-Tool est un ensemble d'outils modulaires pour les développeurs, ingénieurs et passionnés de réseaux. Il permet de réaliser plusieurs opérations comme le scan de réseau, le sniffing de paquets, les requêtes WHOIS, le traceroute, et bien plus.

Modules

Scanner réseau

Ce module scanne une plage d'adresses IP pour identifier les hôtes actifs.

Exemple : IP de départ : 192.168.1.1, IP de fin : 192.168.1.254

Sniffer de paquets

Capture et affiche les paquets réseau en temps réel. Idéal pour le débogage ou l'analyse de trafic réseau.

Exemple : Lancez simplement ce module et surveillez les paquets.

Requête WHOIS

Effectue une recherche WHOIS pour un domaine ou une adresse IP. Ce module est utile pour vérifier l'enregistrement des domaines.

Exemple : Entrez "example.com" pour obtenir des détails.

Traceroute

Affiche le chemin emprunté par les paquets pour atteindre une destination cible. Pratique pour identifier les goulots d'étranglement.

Exemple : traceroute 8.8.8.8

Ping

Envoie des requêtes ICMP pour tester la connectivité avec une cible et mesurer le délai.

Exemple : ping 8.8.8.8

Utilisation

  1. Lancez main.py en exécutant : python main.py.
  2. Suivez le menu pour choisir un module.
  3. Entrez les paramètres demandés pour chaque module.

Installation

Exécutez le fichier install.py pour créer la structure des dossiers et installer les dépendances.

python install.py
