Welcome to Network Multi-Tool

The Network Multi-Tool is a Python project providing essential networking utilities for analyzing and troubleshooting networks.

Packet Sniffer

File: core/packet_sniffer.py

This module captures and displays network packets in real time. It uses libraries like scapy to provide detailed information about each packet.

  • Functions:
    • start_sniffing(): Initiates the packet capturing process.
    • parse_packet(packet): Analyzes the content of a captured packet.
  • Example: 
    python main.py --sniffer

WHOIS Lookup

File: core/whois_lookup.py

This module queries WHOIS information for a given domain or IP address. It retrieves details about ownership and registration.

  • Functions:
    • perform_whois_query(target): Executes a WHOIS query for the specified target.
  • Example: 
    python main.py --whois example.com

Traceroute

File: core/traceroute.py

This module traces the path packets take to reach a target. It identifies intermediate hops and their response times.

  • Functions:
    • perform_traceroute(target): Executes a traceroute for the specified target.
  • Example: 
    python main.py --traceroute example.com

Ping Tool

File: core/ping_tool.py

This module sends ICMP echo requests to test connectivity and measure response times.

  • Functions:
    • ping(target, count=4): Sends a specified number of ping requests to the target.
  • Example: 
    python main.py --ping example.com
