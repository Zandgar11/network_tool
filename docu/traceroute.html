Outil de Traceroute

Le module Traceroute Tool permet de suivre le chemin des paquets à travers le réseau jusqu'à une destination donnée.

Utilisation

Pour utiliser l'outil de traceroute, exécutez le programme principal et sélectionnez l'option correspondante dans le menu CLI.

Fonctionnalités

  • Affiche chaque saut (routeur) entre la source et la destination.
  • Inclut les temps de latence pour chaque saut.
  • Prend en charge les adresses IP et les noms de domaine comme cibles.

Exemple de Code

import subprocess

def traceroute(host):
    """
    Effectue un traceroute vers l'hôte spécifié.
    """
    try:
        result = subprocess.run(["tracert", host], capture_output=True, text=True)
        print(result.stdout)
    except Exception as e:
        print(f"Erreur lors de l'exécution de traceroute : {e}")

Remarque

Le traceroute nécessite des privilèges spécifiques sur certains systèmes pour s'exécuter correctement.
