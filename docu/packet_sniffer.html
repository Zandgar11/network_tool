À propos du Packet Sniffer

Le module Packet Sniffer est conçu pour capturer les paquets réseau en temps réel. Il peut être utilisé pour analyser le trafic réseau, détecter des anomalies ou surveiller l'activité sur un réseau donné.

Fonctionnalités

  • Capture de paquets en direct.
  • Filtrage des paquets par protocole (TCP, UDP, ICMP, etc.).
  • Affichage des informations détaillées sur chaque paquet (source, destination, ports, etc.).

Comment l'utiliser ?

Pour lancer le Packet Sniffer, utilisez la commande suivante :

python main.py --sniffer

Vous pouvez spécifier des options supplémentaires pour filtrer les résultats :

python main.py --sniffer --filter tcp

Exemple de sortie


[INFO] Sniffing started on interface eth0...
[PACKET] Source: 192.168.1.10, Destination: 8.8.8.8, Protocol: ICMP
[PACKET] Source: 192.168.1.11, Destination: 192.168.1.1, Protocol: TCP, Port: 443
