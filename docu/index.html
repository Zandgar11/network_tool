Bienvenue dans la documentation du Network Multi-Tool

Ce projet Python rassemble plusieurs outils réseau essentiels pour analyser, diagnostiquer et résoudre des problèmes de connectivité. Que vous soyez un administrateur système ou un passionné de réseaux, ce logiciel est conçu pour vous.

Objectifs principaux

  • Analyse de paquets réseau en temps réel avec Packet Sniffer.
  • Consultation des informations WHOIS sur des domaines ou adresses IP.
  • Suivi des chemins empruntés par des paquets avec Traceroute.
  • Test de la connectivité avec des requêtes ICMP via l'outil Ping.
  • Organisation modulaire pour une extensibilité facile.

Structure du projet

Voici un aperçu des principaux fichiers et leur rôle :

  • core/ : Contient les modules essentiels (sniffer, ping, traceroute, etc.).
  • ui/ : Fournit une interface CLI pour interagir avec les modules.
  • utils/ : Inclut des fonctions auxiliaires pour le formatage et la gestion des données.
  • main.py : Point d'entrée principal du projet.

Comment commencer ?

Pour exécuter le projet, installez les dépendances nécessaires et lancez l'application :

python main.py

Utilisez les options de ligne de commande pour accéder aux différents modules :

  • --sniffer : Lance le Packet Sniffer.
  • --whois <domaine> : Effectue une recherche WHOIS.
  • --traceroute <cible> : Effectue un traceroute vers une cible.
  • --ping <cible> : Effectue un ping vers une cible.
