Outil de Ping

Le module Ping Tool permet de vérifier la connectivité réseau avec une adresse IP ou un nom de domaine spécifique.

Utilisation

Pour utiliser l'outil de ping, exécutez le programme principal et choisissez l'option correspondante dans le menu CLI.

Fonctionnalités

  • Envoie des requêtes ICMP à l'adresse cible.
  • Affiche les temps de réponse et la perte de paquets éventuelle.
  • Prend en charge les noms de domaine et les adresses IP.

Exemple de Code

import os

def ping(host):
    """
    Envoie une requête ping à un hôte.
    """
    response = os.system(f"ping -c 4 {host}")
    if response == 0:
        print(f"{host} est joignable.")
    else:
        print(f"{host} est injoignable.")

Remarque

Assurez-vous d'avoir les permissions nécessaires pour exécuter des requêtes ICMP sur votre machine.
